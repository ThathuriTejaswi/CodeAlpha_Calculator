# Calculator Web App

## 📖 Overview

The Glassmorphism Calculator Web App is a modern and interactive calculator built using HTML, CSS, and JavaScript. Inspired by contemporary UI trends, the application combines functionality with an elegant glassmorphism design, providing users with a visually appealing and responsive calculation experience.

This project demonstrates the fundamentals of front-end web development, including user interface design, DOM manipulation, event handling, and responsive styling.

---

## 🚀 Features

* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)

* Additional functionalities:

  * Percentage (%) calculations
  * Delete (DEL) last entered character
  * All Clear (AC) reset functionality
  * Decimal number support

* Enhanced User Experience:

  * Modern glassmorphism-inspired interface
  * Gradient background design
  * Interactive button animations
  * Keyboard support for calculations
  * Visual button highlighting on keyboard and mouse interactions
  * Responsive layout for different screen sizes
